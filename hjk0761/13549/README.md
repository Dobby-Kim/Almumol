## 💡 풀이 방법 요약

덱을 통한 BFS 를 사용했습니다.

0 부터 100,000 까지의 좌표를 그래프의 각 노드로 생각한다면,

수빈이의 움직임은 x 노드로부터 x-1, x+1 노드까지 1초, 2*x 노드까지 0초가 걸리는 것입니다.

이를 이용해 n 부터 k 노드까지 BFS 를 수행하고, 각 노드로 도착하는 데에 걸리는 최소 시간을 기록하였습니다.
